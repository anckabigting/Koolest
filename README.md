# Koolest 🧊

An elegant, high-conversion landing page for **Koolest Aircon Services**, engineered with a premium look and an automated booking system. 

This platform showcases specialized air care, handles dynamic maintenance scheduling, and features an integrated escalation management framework to resolve client issues quickly.

## ✨ Features

* **Premium UI/UX:** Built with a modern Aqua Blue, Cream, and Soft Yellow color palette optimized for high visual contrast and user engagement.
* **Typing Slogan Loop:** A custom, pure-CSS typing typewriter animation that infinitely cycles through core business hooks ("Freshen Up", "Book a Clean Breeze", "Book Koolest").
* **Automated Booking Pipeline:** Fully integrated frontend form hooked directly to Google Forms backend tracking via hidden iframes to log schedules without disruptive page reloads.
* **Escalation & Support Management:** Dedicated issue reporting module for immediate customer care and service quality control.
* **Fully Responsive:** Fluid layouts designed to scale beautifully from compact mobile devices to high-resolution desktop screens.

## 🛠️ Tech Stack

* **Frontend:** Semantic HTML5, CSS3 (Custom Variables, Flexbox, Grid, Keyframe Animations)
* **Interactions:** Vanilla JavaScript for state management and async form submission triggers
* **Data Logging:** Google Sheets and Google App Script Backend Integration
